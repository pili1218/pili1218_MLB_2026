v3.4 2026-04-29

        
271-game empirical rule revision. 12-rule framework — 5 rules revised/reclassified, 2 new rules added. R3 reclassified (single signal = ML, skip O/U). R7 narrowed (GVI65+ = O/U OVER only, ML coin flip skip, UNDER hard ban 0%). R9 revised (Wind OUT requires catalyst). R11 new: Slumping SP = +20% O/U. R12 new: conf 55–60 = dead zone (28%). Priority bet checklist added.

        
    
          
  • R3 Reclassified — Single O/U signal = ML bet (75.0%), skip O/U (37.5%). R3 is now an ML rule, not O/U
    • 
          
  • R4 Revised — WP-Override A = ML bet (63%). O/U UNDER requires confirmed xFIP gate; estimated xFIP → ML only
    • 
          
  • R7 Narrowed — GVI≥65: O/U OVER only (58.9%). ML = 50% coin flip (skip). UNDER = 0.0% (hard ban)
    • 
          
  • R8 Downgraded — MCF is caution context only (ML 51.8%, O/U 45.8%). Not actionable standalone; reduce ML stake 25%
    • 
          
  • R9 Revised — Wind OUT standalone = 50.0% O/U (breakeven, pass). Requires PVS>15 or Slumping SP catalyst
    • 
          
  • R10 Confirmed — Conf 60–64 = O/U 63.6% (use). ML at 60–64 = 33.3% (trap, skip ML entirely)
    • 
          
  • R11 NEW — Slumping SP Power Signal — Either SP slumping (RED>+1.5) = +20% O/U accuracy (62%+, n=45). Away 62.5%, home 61.9%. Add as primary O/U signal
    • 
          
  • R12 NEW — Conf 55–60 Dead Zone — O/U only 28.0% (n=25). Treat identically to <50 for O/U: pass all bets. Adjacent bands (50–55=61.2%, 60–64=63.6%) both outperform
    • 
          
  • Priority Bet Checklist — Tier 1 (≥60%): line 9–10 OVER+signal (68.8%), UNDER 8–9+Slumping (60.5%), PVS>15+OVER (61.3%), RCF+OVER (63.3%). Hard skips: no signal (16.3%), conf 55–60 (28%), GVI65+ UNDER (0%)
